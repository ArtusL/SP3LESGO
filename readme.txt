[Changes since last progress check below]

Survive through endless waves of asteroids which get increasly harder
through higher wave counts. You are able to gain money from destroying
them which can be used on various ship upgrades. 

Controls:

WASD for Player Movement

	- WS to move forward/backward
	= AD to rotate ship

Spacebar to shoot bullet

E to access upgrade ship screen
	
	-The letters in square brackets represent the keybinds for the upgrades
	- Q to return to game




Changes:

- Ship now can attract powerups
- Added Triple Shot powerup
- Made upgrade section into a different screen
- Added title screen
